Automated Border Control Implementation

Discover which airports are using Automated Border Control (ABC). The IATA Interactive map indicates whether passengers have to register or not to proceed through ABC:

  • The dark blue pin indicates the existence of a registered traveler program (RTP).
  • The red pin indicates that an electronic machine readable travel document (eMRTD) can be used which does not require prior enrollment.
  • The light blue pin indicates the existence of multiple systems for the use of ABC.
